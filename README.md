# 🍔 Fast Food App

**Fast Food** is an educational Android app that lets you browse restaurant menus, select dishes, and simulate a complete order.



## 🚀 Features
* **Browse restaurants**: Dynamic list of available establishments.
* **Menu management**: Detailed display of dishes with prices and descriptions.
* **Smart shopping cart**: Add/remove items with total price update.
* **Order confirmation**: Summary screen and confirmation of success.

## 🛠️ Architecture & Technologies
The application follows a modular architecture for better maintainability:
* **Language**: Java / Android SDK
* **UI Components**: RecyclerView, Custom Adapters, ConstraintLayout.
* **Data**: Parsing JSON files to simulate a database.
* **Design**: Material Design components.

## 📂 Project Structure
* `model/`: Contains data classes (Restaurant, Menu, Hours).
* `adapters/`: Logic for linking data and lists (RecyclerView).
* `res/layout/`: Optimized XML user interfaces.
